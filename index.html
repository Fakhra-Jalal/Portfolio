🧠 Deep Learning Framework for Real-Time Tissue Classification
Feb 2024 – Oct 2024 | Published in VFAST Journal 2026
Developed deep learning pipeline for real-time tissue classification (fat, muscle, skin) in robotic-assisted surgery. YOLOv10 achieved 22ms inference time with 96.2% mAP@0.5 for surgical precision.
Models Evaluated: GoogLeNet, YOLOv8, YOLOv10
Dataset: 1,800 tissue images with augmentation
Best Results: YOLOv10 → 22ms inference, 96.2% mAP
📸 Proof: Dataset Samples | Model Comparison Chart | Detection Outputs | Publication (DOI: 10.21015/vtse.v14i1.2291)
